Organize your trips here!

This application will make planning your trip, as easy as planning your pizza, not that pizza planning is easy. This app will enable you to Search information online and Save it for future references. Key features of this single page web app are:

  • Better Organization
  • Contextualizing Content Saving
  • Efficient Sharing
  • Remote Collaboration
  • Responsive- Browser and OS independent

Use Tripzza to make better decisions with regards to travel. We are still in beta, so please be nice.

In order for this demo to work, you must enable pop-ups on this site. Dropbox is happy to so pop-ups enabled :)
